# sql-school-libary-book-database
To run this application locally, download the files, extract them and then open your command line to project folder.  From there type 'npm install' to install the project dependencies.  Make sure you already have npm installed.  Once they're been downloaded, type 'npm start' and the application will run on a local server.  To view the application, open your preferred browser and type 'localhost:3000/'.
